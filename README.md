# Веб-платформа для автошколы “LADA” c личным кабинетом обучающегося

Курсовой проект по теме цифровизации образовательного процесса автошколы. Платформа позволяет привлекать новых клиентов через публичный сайт, а также автоматизировать обучение с помощью раздельных личных кабинетов для учеников, инструкторов и администратора.

### 🛠 Стек технологий:
* **Фронтенд:** HTML5, CSS3, JavaScript (адаптивная верстка, интерактивные элементы).
* **Бэкенд:** PHP (обработка сессий, авторизация, логика кабинетов).
* **База данных:** PhpMyAdmin / MySQL (реляционная БД, PDO-запросы).
* **Среда разработки:** Visual Studio Code.

Интерфейс для учащегося, где отображается его текущая категория, статус обучения, прогресс по теории и практике, а также карточки ближайших занятий.

Инструктор может отслеживать список своих учеников, общую посещаемость в процентах, количество проведенных уроков и историю занятий.

Главная панель управления администратора. Показывает общее количество учеников, инструкторов и групп, а также содержит CRUD-таблицу для управления пользователями.

### 12. Кабинет администратора — Список групп
Панель для контроля учебных групп. Администратор может добавлять новые группы, назначать им инструктора, менять статус («обучение» / «завершено») и даты.

Сюда поступают все заявки от пользователей, заполнивших форму на главной странице. После звонка клиенту администратор отмечает заявку как обработанную.

### 14. Кабинет администратора — Новые обращения
Список сообщений, отправленных пользователями через страницу «Контакты». Администратор может ответить клиенту по телефону или email и перенести обращение в историю.

<img src="https://github.com/user-attachments/assets/cd51bef9-0d79-46af-9646-76114bab41d3" width="974" height="352" alt="Новые обращения" />

### 15. Кабинет администратора — Платежи
Финансовый блок, где фиксируются все оплаты за обучение. Доступны функции добавления платежа, выбора способа оплаты (онлайн, карта, перевод) и печати чека.
